### 在 SQL 数据库上创建 Q&A chain和 Agent 的基本方法
概括地说，任何 SQL 链和代理的步骤都是：
```text
1.将问题转换为 SQL 查询：模型将用户输入转换为 SQL 查询。
2.执行SQL查询：执行SQL查询。
3.回答问题：模型使用查询结果响应用户输入
```
